## Projects

### Instagram Prospector
Automated prospecting tool untuk menemukan competitors, vendors, dan potential clients dari Instagram hashtag.

**Location:** `../instagram-scrape/`

**Documentation:** `research/instagram-prospector-method.md`

### Features
- Google Sheets integration
- Automated competitor/vendor/client detection
- Configurable via Setting sheet
- Service Account authentication

### Quick Start

```bash
cd ../instagram-scrape
node scanner.js
```

## Tech Stack

- HTML/CSS (Website)
- Node.js + Playwright (Scraper)
- Google Sheets API (Data storage)
- Google Service Account (Auth)
